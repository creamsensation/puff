# Puff CLI

## Help
> puff help <br>
> puff h <br>

## Generate
### Generate Component
> puff generate component app/name/...go <br>
> puff g c app/name/...go <br>

### Generate Controller
> puff generate controller app/name/...go <br>
> puff g r app/name/...go <br>


### Generate Module
> puff generate module app/name/...go <br>
> puff g m app/name/...go <br>
